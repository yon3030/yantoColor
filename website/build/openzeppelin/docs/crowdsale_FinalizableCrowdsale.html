Extension of Crowdsale where an owner can do extra work after finishing.

Events

Functions

  • finalization

    function finalization () internal

    Can be overridden to add finalization logic. The overriding function should call super.finalization() to ensure the chain of finalization is executed entirely.

  • finalize

    function finalize () public

    Must be called after crowdsale ends, to do some extra finalization work. Calls the contract's finalization function.

    Modifiers:
    onlyOwner
