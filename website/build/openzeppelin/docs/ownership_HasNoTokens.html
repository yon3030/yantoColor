HasNoTokens

contract HasNoTokens

is CanReclaimToken

Source: ownership/HasNoTokens.sol
Author: Remco Bloemen <remco@2π.com>

This blocks incoming ERC223 tokens to prevent accidental loss of tokens. Should tokens (any ERC20Basic compatible) end up in the contract, it allows the owner to reclaim the tokens.
