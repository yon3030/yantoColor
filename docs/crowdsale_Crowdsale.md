<div class="contract-doc"><div class="contract"><h2 class="contract-header"><span class="contract-kind">contract</span> Crowdsale</h2><p class="description">Crowdsale is a base contract for managing a token crowdsale. Crowdsales have a start and end timestamps, where investors can make token purchases and the crowdsale will assign them tokens based on a token per ETH rate.
